# 🎓 Student Performance Prediction System

## 📌 Project Overview
This project is a Machine Learning system designed to predict:

-  Student Final Grades  
-  Student Dropout Risk  

It uses historical academic performance, attendance records, and demographic data to identify students at risk and support early academic intervention.

---

## 🎯 Objectives
- Predict student academic performance using regression models  
- Identify dropout risk using classification models  
- Perform data preprocessing and feature engineering  
- Build dashboard-style visualizations  
- Evaluate model performance using standard ML metrics  

---

## 📊 Dataset
Dataset used: **UCI Student Performance Dataset**

### Files Included:
- student-mat.csv → Math students dataset  
- student-por.csv → Portuguese students dataset  
- student-merge.R → Script to merge datasets  
- student.txt → Dataset description
